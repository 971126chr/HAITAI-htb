Display 속성

box-model 계열

block

콘탠츠를 block 요소로 만들어줌. 항상 줄바꿈 되고 전체 너비 차지함

width & height 지정 가능

div, h1~h6, p, form, header, footer, section,ul, li 등 존재

==>예제
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Etiam semper diam at erat pulvinar, at pulvinar felis blandit.

HELLO WORLD!

Vestibulum volutpat tellus diam, consequat gravida libero rhoncus ut.

inline

콘탠츠를 inline 요소로 만들어줌. 콘탠츠 크기만큼만 자리 차지함. 줄바꿈 없음

크기가 정해져 있어, width & height 값 지정 불가

span, a, img, button 등 존재

==>예제
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Etiam semper diam at erat pulvinar, at pulvinar felis blandit.

HELLO WORLD!

Vestibulum volutpat tellus diam, consequat gravida libero rhoncus ut.

inline-block

콘탠츠가 inline 이면서 block이 됨. 다른 요소들과 같은 줄에 있을 수 있지만 블럭요소임.

width & height 지정 가능

==>예제
Lorem ipsum dolor sit amet, consectetur adipiscing elit. Etiam semper diam at erat pulvinar, at pulvinar felis blandit.

HELLO WORLD!

Vestibulum volutpat tellus diam, consequat gravida libero rhoncus ut.

list 계열

list-item

list가 아닌 태그를 list처럼 바꿔줌, li 태그를 쓰지 않았지만 list같은 레이아웃에 사용

이 속성을 쓴 태그는 li 태그처럼 list-style-type, list-style-position, list-style-image 속성을 이용할 수 있다

==>예제
I will display as a list item

Table (html 상 table 태그)

요소가 block 형식의 table을 나타내도록 바꿔줌

inline-table (html 상 table 태그)

요소가 inline 형식의 table을 나타내도록 바꿔줌

table-row-group (html 상 tbody 태그)

요소가 행(가로줄)을 그룹화하도록 정해줌

table-header-group (html 상 thead 태그)

table 상 header에 해당하는 행을 그룹화해줌
table-header-group이 있는 여러 요소가 포함된 경우 첫번째 요소만 header로 렌더링되고 나머지는 table-row-group으로 처리됨

table-footer-group (html 상 tfoot 태그)

table 상 footer에 해당하는 행을 그룹화해줌
table-footer-group이 있는 여러 요소가 포함된 경우 첫번째 요소만 footer로 렌더링되고 나머지는 table-row-group으로 처리됨

table-row (html 상 tr 태그)

요소가 하나의 행(가로줄) 임을 나타냄

table-column-group (html 상 colgroup 태그)

요소가 열 (세로줄)을 그룹화하게 나타냄

table-column (html 상 col)

요소가 하나의 셀의 열을 설명하도록 나타냄

table-cell (html 상 td, th 태그)

요소가 테이블 셀을 나타내도록 함

table-caption (html 상 caption 태그)

table의 caption을 지정. 즉 표를 설명해주는 제목이 나타남

기타

none

HELLO WORLD!
