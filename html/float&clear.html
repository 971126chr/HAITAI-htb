float 해제 6가지 방법

  1. float된 요소의 부모요소에 강제로 높이 값 지정
  2. float된 요소의 부모요소에 overflow: hidden을 적용
  3. float된 요소의 부모요소에 overflow: auto를 적용
  4. float된 요소의 부모요소에 float 지정
  5. float된 요소의 다음에 오는 태그에 clear: both 지정
  6. 추천! float된 요소의 부모요소에 가상요소(::after 사용)를 만들고 해당 요소에 clear: both 지정

float 해제 방법의 단점

  1. 반응형 시 자동 높이 값 설정 불가능
  2. 해당 요소 속 콘탠츠를 박스 외부로 표현 불가 (잘림)
  3. 특정 브라우저에서는 스크롤 바 나타남
  4. 가운데 배치가 불가능
  5. clear: both가 적용된 요소는 margin-top 적용 불가

가상의 엘리먼트 ::after를 생성 후 display: block; 과 clear: both; 를 추가하면 의미 없는 빈 엘리먼트를 사용하지 않으면서 엘리먼트를 사용해 깔끔하게 float이 clear된다.
